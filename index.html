Métodos de Arreglos en JavaScript

1. `push()` ➕

Descripción: Agrega uno o más elementos al final de un arreglo y devuelve la nueva longitud del arreglo.

Pasos:

  1. Se llama al método `push()` en el arreglo.
  2. Se pasan los elementos que se quieren agregar al final del arreglo.
  3. El método agrega estos elementos al final del arreglo.
  4. Devuelve la nueva longitud del arreglo.
let frutas = ['🍎', '🍌'];
frutas.push('🍇'); // ['🍎', '🍌', '🍇']

2. `pop()` ➖

Descripción: Elimina el último elemento de un arreglo y lo devuelve.

Pasos:

  1. Se llama al método `pop()` en el arreglo.
  2. El método elimina el último elemento del arreglo.
  3. Devuelve el elemento eliminado.
  4. El arreglo se modifica y ahora tiene un elemento menos.
let frutas = ['🍎', '🍌', '🍇'];
let ultimaFruta = frutas.pop(); // ['🍎', '🍌']

3. `shift()` ⬅️

Descripción: Elimina el primer elemento de un arreglo y lo devuelve.

Pasos:

  1. Se llama al método `shift()` en el arreglo.
  2. El método elimina el primer elemento del arreglo.
  3. Devuelve el elemento eliminado.
  4. El arreglo se modifica y ahora el primer elemento es el siguiente en la lista.
let frutas = ['🍎', '🍌', '🍇'];
let primeraFruta = frutas.shift(); // ['🍌', '🍇']

4. `unshift()` ⬆️

Descripción: Agrega uno o más elementos al principio de un arreglo y devuelve la nueva longitud del arreglo.

Pasos:

  1. Se llama al método `unshift()` en el arreglo.
  2. Se pasan los elementos que se quieren agregar al principio del arreglo.
  3. El método agrega estos elementos al principio del arreglo.
  4. Devuelve la nueva longitud del arreglo.
let frutas = ['🍎', '🍌'];
frutas.unshift('🍇'); // ['🍇', '🍎', '🍌']

5. `splice()` ✂️

Descripción: Cambia el contenido de un arreglo eliminando, reemplazando o agregando elementos.

Pasos:

  1. Se llama al método `splice()` en el arreglo.
  2. Se especifica el índice a partir del cual se deben realizar los cambios.
  3. Se especifica el número de elementos que se deben eliminar (si se desea).
  4. Se especifican los elementos a agregar (si se desea).
  5. El método modifica el arreglo original y devuelve una matriz con los elementos eliminados.
let frutas = ['🍎', '🍌', '🍇'];
frutas.splice(1, 1, '🍓', '🍊'); // ['🍎', '🍓', '🍊', '🍇']

6. `slice()` 🍰

Descripción: Devuelve una copia superficial de una porción de un arreglo en un nuevo arreglo seleccionado desde el inicio hasta el final (final no incluido) sin modificar el arreglo original.

Pasos:

  1. Se llama al método `slice()` en el arreglo.
  2. Se especifica el índice de inicio (inclusive) para la copia.
  3. Se especifica el índice de fin (exclusivo) para la copia.
  4. Devuelve un nuevo arreglo que contiene los elementos desde el índice de inicio hasta el índice de fin.
  5. El arreglo original permanece sin cambios.
let frutas = ['🍎', '🍌', '🍇'];
let algunasFrutas = frutas.slice(1, 3); // ['🍌', '🍇']

7. `forEach()` 👀

Descripción: Ejecuta una función proporcionada una vez por cada elemento del arreglo.

Pasos:

  1. Se llama al método `forEach()` en el arreglo.
  2. Se pasa una función como argumento.
  3. El método ejecuta la función para cada elemento del arreglo.
  4. No modifica el arreglo original y no devuelve un valor.
let frutas = ['🍎', '🍌', '🍇'];
frutas.forEach(fruta => console.log(fruta)); // Imprime '🍎', '🍌', '🍇'

8. `map()` 🗺️

Descripción: Crea un nuevo arreglo con los resultados de la llamada a una función proporcionada en cada elemento del arreglo.

Pasos:

  1. Se llama al método `map()` en el arreglo.
  2. Se pasa una función como argumento.
  3. El método ejecuta la función para cada elemento del arreglo.
  4. Devuelve un nuevo arreglo con los resultados de la función.
  5. El arreglo original permanece sin cambios.
let numeros = [1, 2, 3];
let dobles = numeros.map(num => num * 2); // [2, 4, 6]

9. `filter()` 🚦

Descripción: Crea un nuevo arreglo con todos los elementos que pasen la prueba implementada por la función proporcionada.

Pasos:

  1. Se llama al método `filter()` en el arreglo.
  2. Se pasa una función como argumento.
  3. El método ejecuta la función para cada elemento del arreglo.
  4. Devuelve un nuevo arreglo con los elementos que pasaron la prueba.
  5. El arreglo original permanece sin cambios.
let numeros = [1, 2, 3, 4, 5];
let pares = numeros.filter(num => num % 2 === 0); // [2, 4]

10. `find()` 🔍

Descripción: Devuelve el primer elemento del arreglo que cumple con la función de prueba proporcionada.

Pasos:

  1. Se llama al método `find()` en el arreglo.
  2. Se pasa una función como argumento.
  3. El método ejecuta la función para cada elemento del arreglo hasta que encuentra uno que pase la prueba.
  4. Devuelve el primer elemento que pasa la prueba.
  5. Si ningún elemento pasa la prueba, devuelve undefined.
let numeros = [1, 2, 3, 4, 5];
let mayorQueTres = numeros.find(num => num > 3); // 4

11. `findIndex()` 🧭

Descripción: Devuelve el índice del primer elemento del arreglo que cumple con la función de prueba proporcionada.

Pasos:

  1. Se llama al método `findIndex()` en el arreglo.
  2. Se pasa una función como argumento.
  3. El método ejecuta la función para cada elemento del arreglo hasta encontrar uno que pase la prueba.
  4. Devuelve el índice del primer elemento que pasa la prueba.
  5. Si ningún elemento pasa la prueba, devuelve -1.
let numeros = [1, 2, 3, 4, 5];
let indiceMayorQueTres = numeros.findIndex(num => num > 3); // 3

12. `some()` 🤔

Descripción: Prueba si al menos un elemento del arreglo cumple con la función de prueba proporcionada. Devuelve true si al menos uno cumple, y false si ninguno cumple.

Pasos:

  1. Se llama al método `some()` en el arreglo.
  2. Se pasa una función como argumento.
  3. El método ejecuta la función para cada elemento del arreglo.
  4. Si encuentra un elemento que pasa la prueba, devuelve true inmediatamente.
  5. Si ningún elemento pasa la prueba, devuelve false.
let numeros = [1, 2, 3, 4, 5];
let hayMayoresDeTres = numeros.some(num => num > 3); // true

13. `every()` ✅

Descripción: Prueba si todos los elementos del arreglo cumplen con la función de prueba proporcionada. Devuelve true si todos cumplen, y false si al menos uno no cumple.

Pasos:

  1. Se llama al método `every()` en el arreglo.
  2. Se pasa una función como argumento.
  3. El método ejecuta la función para cada elemento del arreglo.
  4. Si todos los elementos pasan la prueba, devuelve true.
  5. Si al menos un elemento no pasa la prueba, devuelve false.
let numeros = [2, 4, 6];
let todosPares = numeros.every(num => num % 2 === 0); // true

14. `reduce()` ➕➡️

Descripción: Ejecuta una función reductora sobre cada elemento del arreglo, resultando en un único valor.

Pasos:

  1. Se llama al método `reduce()` en el arreglo.
  2. Se pasa una función reductora y un valor inicial como argumentos.
  3. La función reductora se ejecuta para cada elemento del arreglo, acumulando un valor.
  4. Devuelve el valor acumulado final.
  5. El arreglo original permanece sin cambios.
let numeros = [1, 2, 3, 4];
let suma = numeros.reduce((acc, num) => acc + num, 0); // 10

15. `sort()` 🔀

Descripción: Ordena los elementos del arreglo en su lugar y devuelve el arreglo ordenado.

Pasos:

  1. Se llama al método `sort()` en el arreglo.
  2. Opcionalmente, se pasa una función de comparación como argumento.
  3. El método ordena los elementos del arreglo según la función de comparación.
  4. Si no se proporciona una función de comparación, los elementos se ordenan como cadenas de texto por defecto.
  5. Devuelve el arreglo ordenado.
  6. El arreglo original se modifica.
let frutas = ['🍎', '🍌', '🍇'];
frutas.sort(); // ['🍌', '🍎', '🍇']

16. `reverse()` 🔄

Descripción: Invierte el orden de los elementos del arreglo en su lugar.

Pasos:

  1. Se llama al método `reverse()` en el arreglo.
  2. El método invierte el orden de los elementos del arreglo.
  3. Devuelve el arreglo invertido.
  4. El arreglo original se modifica.
let frutas = ['🍎', '🍌', '🍇'];
frutas.reverse(); // ['🍇', '🍌', '🍎']

17. `findIndex()` 🧭

Descripción: Devuelve el índice del primer elemento que cumple con la función de prueba proporcionada.

Pasos:

  1. Se llama al método `findIndex()` en el arreglo.
  2. Se pasa una función como argumento.
  3. El método ejecuta la función para cada elemento del arreglo hasta encontrar uno que pase la prueba.
  4. Devuelve el índice del primer elemento que pasa la prueba.
  5. Si ningún elemento pasa la prueba, devuelve -1.
let numeros = [1, 2, 3, 4, 5];
let indiceMayorQueTres = numeros.findIndex(num => num > 3); // 3

18. `Destructuring` 🧩

Descripción: Permite extraer valores de arreglos o propiedades de objetos en variables distintas.

Pasos:

  1. Se usa una sintaxis especial para extraer valores de un arreglo u objeto.
  2. Para arreglos, se colocan corchetes alrededor de las variables a asignar.
  3. Para objetos, se colocan llaves alrededor de las variables a asignar.
  4. Los valores se asignan en el orden de los elementos del arreglo o las propiedades del objeto.
let frutas = ['🍎', '🍌', '🍇'];
let [manzana, platano, uva] = frutas; // manzana = '🍎', platano = '🍌', uva = '🍇'

19. Operador Rest (...) 📦

Descripción: Permite representar un número indefinido de argumentos como un arreglo.

Pasos:

  1. Se usa el operador ... antes de una variable en una función o declaración de arreglo.
  2. En una función, captura todos los argumentos restantes en un solo arreglo.
  3. En una declaración de arreglo, captura todos los elementos restantes en un solo arreglo.
function sumar(...numeros) {
    return numeros.reduce((acc, num) => acc + num, 0);
}
sumar(1, 2, 3, 4); // 10

20. Operador Spread (...) 🌌

Descripción: Permite expandir elementos de un arreglo u objeto en lugares donde se esperan múltiples elementos.

Pasos:

  1. Se usa el operador ... antes de un arreglo u objeto.
  2. En un arreglo, expande los elementos individuales en un nuevo contexto.
  3. En un objeto, expande las propiedades individuales en un nuevo contexto.
let frutas = ['🍎', '🍌', '🍇'];
let otrasFrutas = ['🍓', ...frutas, '🍊']; // ['🍓', '🍎', '🍌', '🍇', '🍊']
